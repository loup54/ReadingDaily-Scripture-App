## Legal Documents Guide

### Viewing Legal Documents

#### Getting Started

1. Open the app and navigate to **Settings**
2. Tap **Legal & Compliance**
3. Select **Legal Documents**

#### Available Documents

The app includes the following legal documents:

- **Terms of Service**: Complete terms governing app usage
- **Privacy Policy**: How we collect and use your data
- **Data Processing Terms**: Technical details about data handling

#### Reading Documents

**Viewing a Document:**
1. Tap on any document in the list
2. The document viewer opens with the full content
3. Content is displayed in expandable sections for easy reading

**Expanding Sections:**
- Tap any section header to expand/collapse
- Tap **Expand All** to see all content at once
- Tap **Collapse All** to hide all section content

**Searching Documents:**
1. Use the search bar at the top
2. Enter keywords or phrases (e.g., "warranty", "privacy")
3. Results highlight matching sections
4. Tap the X to clear search

**Sharing Documents:**
1. Tap the **Share** button (arrow icon)
2. Choose how to share (email, messaging, etc.)
3. Document details are sent to your chosen app

#### Document Metadata

At the bottom of each document, you'll see:
- **Number of sections**: Count of document sections
- **Last Updated**: Date of most recent changes
- **Effective Date**: When the document became active
- **Questions?**: Contact email for legal inquiries

### Accepting Documents

Some documents require acceptance (indicated by a blue button):

**To Accept Without Signature:**
1. Read through the document
2. Tap **I Agree & Accept** at the bottom
3. Confirmation appears
4. Document marked as accepted

**To Accept With Signature (when required):**
1. Read through the document
2. Tap **I Agree & Sign** at the bottom
3. Signature modal appears with two options:
   - **Draw Signature**: Sketch your signature
   - **Type Name**: Type your full name
4. Complete signature
5. Tap **Done**
6. Signature saved and document marked as accepted

#### Signature Options

**Sketch Signature (Recommended for authenticity):**
- Draw your signature on the canvas
- Use your finger or stylus
- Minimum 1 second of drawing
- Tap **Done** to save

**Typed Signature (Quick & accessible):**
- Type your full name in the text field
- Select the style (script or print)
- Tap **Done** to save
- Better for accessibility needs

### Document History

Your document acceptance history is saved and includes:
- Date and time accepted
- Document version at time of acceptance
- Whether it was signed
- Device type (iPhone/Android) and app version

**View History:**
1. Go to Settings → Compliance & Analytics
2. Select the Timeline tab
3. See all past acceptances with dates

## Backup & Export Guide

### Creating Backups

#### Local Backups

**Step-by-Step:**

1. **Open Backup Screen**
   - Go to Settings → Legal & Compliance → Backup & Export
   - Select the **Backup** tab (default)

2. **Create Backup**
   - Tap **Create Backup Now** button
   - App asks: "Protect with password?"

3. **Without Password (Simple):**
   - Tap **No, Skip**
   - Backup created in seconds
   - See it listed in backup history

4. **With Password (Recommended):**
   - Tap **Yes, Set Password**
   - Password modal appears
   - Enter password (minimum 6 characters)
   - Confirm password (must match)
   - Tap **Create Protected Backup**
   - Backup created and encrypted

#### What's Backed Up?

Each backup includes:
- All legal documents
- Your acceptances and signatures
- Compliance reports and analytics
- Audit trail (complete history)
- Settings and preferences

#### Backup Status

**Status Information:**
- **Last Backup**: When your most recent backup was created
- **Next Backup**: Scheduled time for auto-backup
- **Total Backups**: How many backups you have saved
- **Auto Backup**: Whether monthly automatic backups are enabled

### Cloud Backups

#### Uploading to Cloud

**Automatic Cloud Sync:**
1. Enable auto-backup (see "Auto Backup" setting)
2. Select monthly schedule (default)
3. App automatically backs up monthly to cloud
4. Backups encrypted and secured

**Manual Upload:**
1. Create local backup first
2. Open Backup & Export
3. Tap **History** tab
4. Tap the **Upload** button on your backup
5. Choose to protect with password (if not already protected)
6. Backup uploads to cloud
7. Shows "Synced" status when complete

**Why Cloud Backup?**
- **Safety**: Data safe if phone is lost
- **Access**: Restore on any device
- **Security**: End-to-end encrypted
- **Automatic**: Happens monthly without action

### Restoring Data

#### From Local Backup

1. Open Backup & Export → **Restore** tab
2. See list of local backups
3. Each shows: date, size, document count
4. Tap the **Restore** button on desired backup
5. Confirmation alert appears
6. If password-protected, enter password
7. Tap **Restore**
8. Data restored (overwrites current data)
9. Takes 10-30 seconds

#### From Cloud Backup

1. Open Backup & Export → **History** tab
2. See "Cloud Backups" section
3. Tap **Restore** on desired cloud backup
4. If encrypted, enter password when prompted
5. Tap **Restore**
6. Downloads and restores data
7. Takes 30-60 seconds (depends on speed)

**⚠️ Important:** Restore overwrites all current data. Make sure you want to replace your current data before confirming.

### Managing Backups

**Viewing Backup Details:**
- Backup date and time
- Size in MB
- Number of documents
- Number of acceptances
- Number of signatures
- Verification status (✓ Verified)

**Deleting Backups:**
1. Tap the **Delete** (trash) button
2. Confirm deletion
3. Backup removed

**Best Practices:**
- Create backups monthly
- Keep at least 3 recent backups
- Delete very old backups to save space
- Use cloud backups for long-term storage

### Exporting for Sharing

#### Export to File

1. Open Backup & Export → **Export** tab
2. Tap **Export as ZIP**
3. Creates portable backup file
4. File ready to save or share

#### Sharing Exports

**Via Email:**
1. Export document (JSON, CSV, or PDF)
2. Tap Share button
3. Select Email
4. Attaches to new email
5. Send to recipient

**Via Cloud Storage:**
1. Export document
2. Use Files app or cloud service
3. Upload to Google Drive, Dropbox, iCloud, etc.
4. Share link with others

**Via AirDrop (iOS):**
1. Export file
2. Select AirDrop from share options
3. Select recipient nearby
4. File transfers wirelessly

### Auto-Backup Setup

#### Enabling Auto-Backup

1. Open Backup & Export → **Backup** tab
2. Find **Auto Backup** toggle
3. Switch ON
4. Tap to enable: "Auto backup enabled - monthly schedule set"
5. System sets up monthly backups

#### How Auto-Backup Works

- **Frequency**: Monthly (same date each month)
- **Time**: Automatic (can happen anytime)
- **Destination**: Cloud storage
- **Encryption**: Uses your phone's security
- **Retry**: Attempts up to 3 times if it fails
- **Notification**: You'll see confirmation when complete

#### Auto-Backup Settings

**Current Settings:**
- Next scheduled backup date shown
- Days until next backup calculated
- Last backup time displayed
- Total backups stored shown

**Stopping Auto-Backup:**
1. Open Backup & Export
2. Toggle **Auto Backup** OFF
3. No more automatic monthly backups
4. You can still create manual backups

### Password Protection

#### Setting Passwords

**Requirements:**
- Minimum 6 characters
- Can use letters, numbers, and symbols
- Recommended: Mix of uppercase, lowercase, numbers

**Example:**
- ✓ SecurePass123!
- ✓ MyDoc@2024
- ✗ 12345 (too simple)

**Best Practices:**
- Use strong, unique passwords
- Don't share passwords
- Remember your password
- Write it down in a secure location (optional)

**If You Forget Password:**
- Backup becomes inaccessible
- You'll need to create a new backup
- Cannot recover forgotten passwords

#### Using Protected Backups

1. When restoring encrypted backup
2. Password prompt appears
3. Enter the same password used at creation
4. Click "Validate"
5. Shows ✓ Correct when matched
6. Backup restores successfully

## Troubleshooting

### Common Issues & Solutions

#### Documents Won't Open

**Problem:** Document fails to load or shows error

**Solutions:**
1. Check internet connection
2. Close and reopen the app
3. Force quit app and restart
4. Check if storage space is available
5. Update app to latest version

#### Can't Create Backup

**Problem:** Backup creation fails or hangs

**Solutions:**
1. Ensure sufficient storage space (500MB+ free)
2. Check internet connection
3. Close other running apps
4. Restart your device
5. Try again in a few minutes

#### Forgot Backup Password

**Problem:** Cannot restore password-protected backup

**Solutions:**
1. Password cannot be recovered
2. Create a new backup with new password
3. Contact support for assistance
4. Consider using Face ID/Touch ID instead

#### Signature Won't Save

**Problem:** Signature capture fails

**Solutions:**
1. Ensure good lighting
2. Keep hand steady
3. Avoid pressing too hard
4. Try typed signature instead
5. Update app to latest version

## Glossary

**Acceptance**: Agreeing to a legal document

**Audit Trail**: Complete record of all actions taken

**Backup**: Copy of your data for safekeeping

**Cloud**: Remote internet-based storage

**Compliance**: Meeting legal requirements

**Encryption**: Scrambling data so only you can access it

**Jurisdiction**: Geographic area with specific laws

**Signature**: Digital representation of your agreement

**Sync**: Automatically updating between devices

**Timestamp**: Date and time when something happened
