# Hero Image Instructions

## Required Image

**File:** `hero-bible.jpg` or `hero-bible.png`

**Specifications:**
- Dimensions: 800x560px (or similar 4:3 ratio)
- Format: JPG or PNG
- Content: Open Bible with golden/warm lighting
- Reference: See the golden Bible image from the design mockup

## How to Add

1. Save your Bible image as `hero-bible.jpg` in this folder
2. The LandingScreen will automatically display it
